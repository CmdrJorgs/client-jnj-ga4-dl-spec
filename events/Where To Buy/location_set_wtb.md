# Location Set WTB

Fire whenever a user successfully sets thier location. 

## Javascript Code

```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({ event_data: null, user_data: null });  // Clear the previous event_data object.
dataLayer.push({
  event: 'location_set_wtb',
  },
});
```
